Below, you can download the two example projects used in the paper to illustrate the debugger:

  • Draw The Star. In this project, the player (the drawer) should follow the path and not touch the water.
  • The Maze. In this project, the player must navigate a maze.

Open a Scratch Editor with debugger support.

To enable the debugger, press the "Bug" icon next to the built-in Scratch controls (the green flag and red stop button). Once enabled (indicated by a green top bar), you can use the following features:

  • Pause blocks (or breakpoints).
  • Pause or resume execution.
  • Execute step by step when paused.
  • Use the timeline to go back to previous states.

Source code

The source code for the debugger is available at github.com/scratch4d/scratch-debugger.
